⚛️ TP React : Création d'une application avec Vite

Ce travail pratique vous guidera à travers l'installation des outils nécessaires et la création d'une application React moderne utilisant Vite. Vous apprendrez à configurer un environnement de développement complet et à comprendre la structure d'un projet React.

🎯 Objectifs de ce TP :
  • Installer et configurer Visual Studio Code
  • Installer Node.js et npm (gestionnaire de paquets)
  • Utiliser le terminal en ligne de commande
  • Créer une application React avec Vite
  • Comprendre la structure d'un projet React
  • Découvrir le rôle des fichiers essentiels
  • Documenter son travail avec des captures d'écran

📦 ÉTAPE 1 : Installation des outils de développement

1.1 - Téléchargement et installation de Visual Studio Code

💻 Installer l'éditeur de code officiel pour le développement React

Visual Studio Code est l'IDE le plus populaire pour le développement JavaScript/React grâce à ses nombreuses extensions.

📌 Procédure d'installation :
  1. Rendez-vous sur le site officiel : https://code.visualstudio.com/
  2. Téléchargez la version correspondant à votre système d'exploitation (Windows, macOS, Linux)
  3. Lancez l'installateur et suivez les instructions
  4. Important (Windows) : Cochez les options :
    • "Ajouter l'action 'Ouvrir avec Code' au menu contextuel"
    • "Ajouter à PATH" (pour utiliser la commande 'code' dans le terminal)
  5. Une fois installé, lancez VS Code

1.2 - Téléchargement et installation de Node.js et npm

🟢 Installer l'environnement d'exécution JavaScript et son gestionnaire de paquets

Node.js est indispensable pour exécuter des applications JavaScript côté serveur et utiliser npm (Node Package Manager) pour installer les bibliothèques nécessaires comme React.

📌 Procédure d'installation :
  1. Rendez-vous sur https://nodejs.org/
  2. Téléchargez la version LTS (Long Term Support) - la plus stable
  3. Lancez l'installateur et suivez les instructions par défaut
  4. L'installation inclut automatiquement npm (Node Package Manager)
  5. Redémarrez votre terminal après l'installation

📸 Étape 2 - Capture d'écran

Prenez une capture d'écran du terminal après avoir vérifié les versions de Node.js et npm.

# Vérifiez que Node.js est correctement installé :
$ node --version
v20.11.0 <-- Doit afficher une version récente

# Vérifiez la version de npm :
$ npm --version
10.2.4 <-- Doit afficher une version
ℹ️ À savoir : npm est le gestionnaire de paquets officiel de Node.js. Il permet d'installer des bibliothèques JavaScript comme React, Vite, etc. La commande npx (inclus avec npm) permet d'exécuter des paquets sans les installer globalement.

3.2 - Définition du rôle des fichiers essentiels

📖 Comprendre l'architecture d'une application React avec Vite
📁 Structure du projet reactappvite :
  • 📁 node_modules/ → Dépendances du projet (ne jamais modifier manuellement)
  • 📁 public/ → Fichiers statiques (images, favicon, etc.)
  • 📁 src/ → Code source de l'application
    • 📄 src/main.jsx → Point d'entrée de l'application
    • 📄 src/App.jsx → Composant principal de l'application
    • 🎨 src/App.css → Styles du composant App
    • 🎨 src/index.css → Styles globaux
  • 📄 index.html → Page HTML principale (point d'entrée du navigateur)
  • 📄 package.json → Configuration du projet et dépendances
  • 📄 vite.config.js → Configuration de Vite (bundler)
  • 📄 .gitignore → Fichiers exclus du versionnement Git

  </React.StrictMode>

📄 src/App.jsx

import { useState } from 'react' // Hook pour gérer l'état
import reactLogo from './assets/react.svg'
import viteLogo from '/vite.svg'
import './App.css'

function App() {
  const [count, setCount] = useState(0) // État du compteur

  return (
    <div className="App">
      <div>
        <a href="https://vitejs.dev" target="_blank">
          <img src={viteLogo} className="logo" alt="Vite logo" />
        </a>
        <a href="https://react.dev" target="_blank">
          <img src={reactLogo} className="logo react" alt="React logo" />
        </a>
      </div>
      <h1>Vite + React</h1>
      <div className="card">
        <button onClick={() => setCount((count) => count + 1)}>
          count is {count}
        </button>
      </div>
    </div>
  )
}

export default App

5.1 - Implémenter un compteur de likes interactif

🧮 Dans le composant Home.jsx, ajoutez un compteur de "j'aime" (likes) avec un bouton.
📌 Utilisez le hook useState de React pour gérer l'état du compteur.
🎨 Décorez le bouton et l'affichage du compteur avec Tailwind CSS.

Compteur de j'aime

Cliquez sur le bouton pour incrémenter le compteur

📄 Code à ajouter dans Home.jsx (intégration du useState) : import React, { useState } from 'react'; function Home() { // Déclaration d'un état "likes" avec une valeur initiale à 0 const [likes, setLikes] = useState(0); // Fonction pour incrémenter le nombre de likes const handleLike = () => { setLikes(likes + 1); }; return ( <div className="min-h-screen bg-gradient-to-br from-blue-50 to-indigo-100 flex items-center justify-center p-8"> <div className="max-w-3xl mx-auto bg-white rounded-2xl shadow-xl overflow-hidden"> <div className="bg-gradient-to-r from-blue-600 to-indigo-600 p-6 text-center"> <h1 className="text-4xl font-bold text-white mb-2">Bienvenue sur React + Vite</h1> <h2 className="text-xl text-blue-100">Avec Tailwind CSS et useState</h2> </div> <div className="p-8"> <div className="bg-gray-50 rounded-lg p-6 border-l-4 border-blue-500 mb-6"> <p className="text-gray-700 text-lg leading-relaxed"> Ce composant Home a été créé spécialement pour ce TP. 🧠 Explication du useState :
  • const [likes, setLikes] = useState(0); → Déclare une variable d'état likes initialisée à 0, et une fonction setLikes pour la modifier.
  • Lorsque l'utilisateur clique sur le bouton, setLikes(likes + 1) est appelé, ce qui met à jour l'état et provoque un re-rendu du composant avec la nouvelle valeur.
  • React gère automatiquement la mise à jour de l'interface utilisateur.

📋 Récapitulatif des commandes essentielles :
  • npm create vite@latest nom-app -- --template react → Crée une nouvelle app React
  • cd nom-app → Se déplace dans le dossier du projet
  • npm install → Installe toutes les dépendances listées dans package.json
  • npm install -D tailwindcss postcss autoprefixer → Installe Tailwind CSS
  • npx tailwindcss init -p → Initialise la configuration Tailwind
  • npm run dev → Lance le serveur de développement (http://localhost:5173)
  • npm run build → Compile l'application pour la production (dossier dist/)
⚠️ Dépannage rapide :
  • Erreur "command not found" → Node.js n'est pas installé ou pas dans le PATH. Réinstallez Node.js.
  • Le port 5173 est déjà utilisé → Vite utilise automatiquement le prochain port disponible (5174, 5175...)
  • Tailwind ne s'affiche pas → Vérifiez que vous avez bien configuré les fichiers tailwind.config.js et index.css (ajout des directives @tailwind).
  • Les modifications ne s'affichent pas → Rafraîchissez la page ou redémarrez le serveur (Ctrl+C puis npm run dev)
