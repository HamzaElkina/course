📱 TP React Native : Création d'une première application mobile

Ce travail pratique vous guidera à travers les concepts fondamentaux de React Native. Vous allez créer votre première application mobile fonctionnelle avec Expo, un framework qui simplifie le développement React Native.

🛠️ Prérequis nécessaires

Avant de commencer, assurez-vous d'avoir installé les outils suivants :

📦 Node.js

Version 16 ou supérieure nécessaire pour React Native

Télécharger Node.js

📱 Expo Go

Application mobile pour tester sur votre smartphone

Télécharger Expo Go

💻 VS Code

IDE recommandé pour le développement React Native

Télécharger VS Code
🎯 Objectifs de ce TP :
  • Installer et configurer un projet React Native avec Expo
  • Comprendre la structure d'une application React Native
  • Créer des composants fonctionnels avec JSX
  • Gérer l'état avec useState
  • Ajouter des interactions tactiles et des événements
  • Lancer l'application sur un émulateur ou un appareil physique

Exercice 1

🎯 Créer un nouveau projet React Native avec Expo

La première étape consiste à créer un projet React Native en utilisant Expo.

📌 Indications :
  • Ouvrez votre terminal (invité de commandes ou PowerShell)
  • Exécutez la commande : npx create-expo-app MonPremierProjet
  • Attendez que l'installation des dépendances se termine
  • Naviguez dans le dossier : cd MonPremierProjet
  • Ouvrez le dossier dans VS Code : code .
💡 À faire : Créez votre projet et explorez sa structure de dossiers.
📝 Commandes à exécuter :

npx create-expo-app MaPremiereApp
cd MaPremiereApp
code .

Exercice 2

🎯 Lancer l'application avec npx expo start

Découvrez comment démarrer votre application et la visualiser sur votre appareil.

📌 Indications :
  • Dans le terminal (assurez-vous d'être dans le dossier du projet)
  • Lancez la commande : npx expo start
  • Attendez que le Metro Bundler se lance (un QR code apparaît dans le terminal)
  • Sur votre smartphone : Ouvrez l'application Expo Go
  • Scannez le QR code avec l'application Expo Go
  • L'application devrait s'afficher sur votre téléphone !
💡 À faire : Lancez l'application et vérifiez qu'elle fonctionne correctement.
⚠️ Astuce : Assurez-vous que votre ordinateur et votre smartphone sont sur le même réseau Wi-Fi. Si des problèmes persistent, utilisez l'option "Tunnel" en appuyant sur 't' dans le terminal.
🎮 Commandes utiles pendant l'exécution :

• a - Lancer sur Android (émulateur)
• i - Lancer sur iOS (simulateur)
• w - Lancer sur navigateur Web
• r - Recharger l'application
• m - Ouvrir le menu développeur

Exercice 3

🎯 Analyser la structure du projet et comprendre App.js

Explorez les fichiers générés automatiquement et modifiez le contenu de l'application.

📌 Indications :
  • Ouvrez le fichier App.js dans VS Code
  • Observez la structure : imports, composant fonctionnel, export
  • Identifiez les éléments JSX : <View>, <Text>, <StatusBar>
  • Copiez le code ci-dessous pour remplacer le contenu par défaut
📄 Fichier : App.js (version simplifiée)

import { StatusBar } from 'expo-status-bar';
import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text>Ma première application React Native !</Text>
      <StatusBar style="auto" />
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center',
  },
});
💡 À faire : Remplacez le contenu de App.js par ce code, sauvegardez et observez les changements sur votre téléphone (rechargement automatique).

Exercice 4

🎯 Créer un composant personnalisé "WelcomeCard"

Apprenez à créer des composants réutilisables avec des propriétés (props).

📌 Indications :
  • Créez un nouveau dossier components à la racine du projet
  • Créez un fichier WelcomeCard.js dans ce dossier
  • Créez un composant qui affiche une carte avec nom et description
  • Importez et utilisez ce composant dans App.js
📄 Fichier : components/WelcomeCard.js

import { View, Text, StyleSheet } from 'react-native';

export default function WelcomeCard({ name, description }) {
  return (
    <View style={styles.card}>
      <Text style={styles.name}>👋 {name}</Text>
      <Text style={styles.description}>{description}</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  card: {
    backgroundColor: '#f8f9fa',
    padding: 20,
    borderRadius: 10,
    margin: 15,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
    shadowRadius: 4,
    elevation: 3,
  },
  name: { fontSize: 20, fontWeight: 'bold', marginBottom: 5 },
  description: { fontSize: 14, color: '#666' },
});
📄 Mise à jour de App.js (dans le return)

import WelcomeCard from './components/WelcomeCard';

// Dans le return :
<WelcomeCard
  name="Étudiant React Native"
  description="Découvrez la puissance du développement mobile avec React Native !"
/>
💡 À faire : Créez ce composant et utilisez-le dans votre application avec vos propres informations.

Exercice 5

🎯 Ajouter un compteur avec useState et un bouton tactile

Apprenez à gérer l'état local d'un composant avec le hook useState.

📌 Indications :
  • Importez useState depuis 'react-native'
  • Ajoutez un état count initialisé à 0
  • Créez une fonction incrementCount qui augmente le compteur
  • Ajoutez un composant TouchableOpacity ou Button
  • Affichez la valeur du compteur
📄 Code à ajouter dans App.js

import { ..., TouchableOpacity, Button } from 'react-native';

const [count, setCount] = useState(0);

const incrementCount = () => {
  setCount(count + 1);
};

// Dans le JSX (après WelcomeCard) :
<View style={styles.counterContainer}>
  <Text style={styles.countText}>Compteur : {count}</Text>
  <TouchableOpacity style={styles.button} onPress={incrementCount}>
    <Text style={styles.buttonText}>Appuyez-moi ! 🎯</Text>
  </TouchableOpacity>
</View>
💡 À faire : Ajoutez ce compteur interactif. Testez-le sur votre appareil en appuyant sur le bouton.

Exercice 6 - Extra

🎯 Générer un build de partage avec Expo

Apprenez à créer un fichier APK ou IPA pour partager votre application.

📌 Indications :
  • Installez EAS CLI : npm install -g eas-cli
  • Connectez-vous : eas login
  • Configurez le projet : eas build:configure
  • Générez un build Android : eas build -p android --profile preview
  • Téléchargez et partagez votre fichier APK
📱 Commandes pour générer un APK :

npm install -g eas-cli
eas login
eas build:configure
eas build -p android --profile preview

⚠️ Problèmes courants et solutions :
  • Erreur de connexion : Vérifiez que votre téléphone et ordinateur sont sur le même Wi-Fi
  • Metro Bundler ne démarre pas : Fermez le terminal et réessayez npx expo start --tunnel
  • Erreur de dépendances : Supprimez node_modules et package-lock.json, puis npm install
  • Build échoue : Vérifiez votre connexion internet et réessayez avec eas build --clear-cache
  • Le QR code ne scanne pas : Utilisez l'option 't' dans le terminal pour passer en mode tunnel
