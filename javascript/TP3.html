TP JavaScript : POO, JSON & API avec XHR

Ce travail pratique vous guide à travers les concepts avancés : Programmation Orientée Objet, manipulation JSON, et appel d'API avec XMLHttpRequest. Vous apprendrez également à tester une API avec Postman.

🎯 Objectifs de ce TP :
  • Créer une classe JavaScript avec constructeur, getters et setters
  • Instancier des objets à partir d'une classe
  • Créer et manipuler des objets JSON
  • Tester une API REST avec Postman
  • Utiliser XMLHttpRequest (XHR) pour appeler une API
  • Afficher les données dans un tableau HTML dynamique

Exercice 3

🎯 Tester l'API avec Postman et l'appeler avec XHR pour afficher les prix Bitcoin

Vous allez d'abord tester l'API avec Postman, puis utiliser XMLHttpRequest pour récupérer les données et les afficher dans un tableau.

📬 Étapes pour tester l'API avec Postman :

  1. Ouvrir Postman (télécharger sur postman.com)
  2. Cliquer sur New Request ou +
  3. Méthode : GET
  4. URL : https://mempool.space/api/v1/prices
  5. Cliquer sur Send
  6. Vous devriez voir la réponse JSON :
    {"USD":76074, "EUR":64944, "GBP":56310, ...}
  7. Vérifier le statut de la réponse : 200 OK
💡 Astuce : Postman permet de voir la structure exacte des données reçues, ce qui facilite l'écriture du code JavaScript pour les manipuler.
📌 Indications pour l'appel XHR :
  • Utilisez new XMLHttpRequest() pour créer une requête
  • Méthode : GET vers https://mempool.space/api/v1/prices
  • Utilisez onreadystatechange pour gérer la réponse
  • Vérifiez readyState === 4 et status === 200
  • Parsez la réponse JSON avec JSON.parse()
  • Affichez les prix dans un tableau HTML
  • Gérez les erreurs (timeout, réseau, etc.)
💡 À faire : Cliquez sur le bouton ci-dessous pour charger les prix Bitcoin avec XHR et les afficher dans un tableau.
⚡ Cliquez sur le bouton pour charger les données
// Exemple de code XHR à écrire :

const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://mempool.space/api/v1/prices', true);
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4) {
        if (xhr.status === 200) {
            const data = JSON.parse(xhr.responseText);
            // Afficher dans un tableau
        }
    }
};
xhr.send();
🔍 Outils importants :
  • Postman : Testez l'API avant de coder pour comprendre la structure des données
  • Console navigateur (F12) : Visualisez les requêtes XHR et les erreurs
  • Onglet Network : Voir les requêtes HTTP et les réponses
