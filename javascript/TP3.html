TP JavaScript : POO, JSON & API

Ce travail pratique vous guidera à travers les concepts avancés de JavaScript : Programmation Orientée Objet, manipulation du JSON, et appel à une API externe. N'oubliez pas d'utiliser la console du navigateur (F12) pour tester et déboguer votre code.

🎯 Objectifs de ce TP :
  • Créer une classe JavaScript avec constructeur, getters et setters
  • Instancier des objets à partir d'une classe
  • Créer et manipuler des objets JSON
  • Utiliser l'API Fetch pour récupérer des données en temps réel
  • Afficher dynamiquement des données dans la page HTML

Exercice 1

🎯 Créer une classe "Personne" avec getters et setters, puis l'instancier

Vous allez créer une classe JavaScript représentant une personne, avec des propriétés privées, des getters et des setters, puis instancier cette classe.

📌 Indications :
  • Créez une classe Personne avec un constructeur prenant nom, prenom et age
  • Utilisez des propriétés privées (avec #) ou des conventions (ex: _nom)
  • Créez des getters : get nom(), get prenom(), get age()
  • Créez des setters : set nom(valeur), set prenom(valeur), set age(valeur)
  • Dans le setter de l'âge, vérifiez que l'âge est un nombre positif
  • Ajoutez une méthode sePresenter() qui retourne une phrase de présentation
  • Instanciez un objet personne1 avec vos propres informations
  • Affichez le résultat dans la console et dans la page HTML
💡 À faire : Écrivez la classe Personne, instanciez-la, et affichez sa présentation.
// À écrire dans votre fichier script.js ou dans une balise <script>

class Personne {
    #nom;
    #prenom;
    #age;

    constructor(nom, prenom, age) {
        this.nom = nom;
        this.prenom = prenom;
        this.age = age;
    }

    // Getters
    get nom() { return this.#nom; }
    // Setters avec validation
    set nom(valeur) { this.#nom = valeur; }
    // ... à compléter
    sePresenter() { ... }
}

Exercice 2

🎯 Créer un objet JSON décrivant une personne (nom, prénom, âge)

Vous allez créer un objet au format JSON, le manipuler, le convertir en chaîne et le parser.

📌 Indications :
  • Créez un objet JavaScript classique avec les propriétés : nom, prenom, age
  • Convertissez cet objet en chaîne JSON avec JSON.stringify()
  • Affichez la chaîne JSON dans la console
  • Reconvertissez la chaîne en objet JavaScript avec JSON.parse()
  • Ajoutez une nouvelle propriété "ville" à l'objet et affichez le JSON modifié
  • Affichez les résultats dans la page HTML
💡 À faire : Manipulez le format JSON pour comprendre sa structure.
// Exemple de manipulation JSON :

const personne = {
    nom: "Dupont",
    prenom: "Marie",
    age: 28
};

const jsonString = JSON.stringify(personne);
console.log("JSON :", jsonString);
const parsed = JSON.parse(jsonString);
console.log("Objet reconstruit :", parsed);

Exercice 3

🎯 Utiliser l'API Mempool pour afficher la valeur du Bitcoin selon la devise

Vous allez appeler une API publique qui donne le prix du Bitcoin en temps réel, et afficher le prix dans différentes devises.

📌 Indications :
  • API à utiliser : https://mempool.space/api/v1/prices
  • Cette API renvoie un objet JSON avec les prix en USD, EUR, GBP, CAD, CHF, AUD, JPY
  • Utilisez fetch() pour récupérer les données
  • Affichez le prix en USD par défaut
  • Créez une liste déroulante (<select>) pour choisir la devise
  • Affichez le prix mis à jour lorsque l'utilisateur change de devise
  • Gérez les erreurs (API inaccessible, etc.)
  • Affichez un message de chargement pendant la récupération des données
💡 À faire : Afficher le prix du Bitcoin en temps réel avec sélecteur de devise.
// Exemple d'appel API avec fetch :

async function getBitcoinPrice() {
    try {
        const response = await fetch('https://mempool.space/api/v1/prices');
        const data = await response.json();
        console.log(data);
        // {USD: 76074, EUR: 64944, ...}
    } catch (error) {
        console.error("Erreur:", error);
    }
}
