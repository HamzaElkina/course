📌 TP2 JavaScript : DOM & Événements

Ce TP explore la manipulation du DOM : récupération, modification, insertion, suppression d'éléments, et gestion d'événements (click, keyup, focus, submit). Ouvrez la console (F12) pour observer les logs.

🎯 Objectifs :
  • Utiliser getElementById, querySelector, getElementsByClassName
  • Modifier contenu, attributs, styles CSS
  • Créer et insérer des éléments (createElement, appendChild, insertAdjacentHTML)
  • Supprimer des éléments (remove(), removeChild)
  • Gérer événements : click, keyup, focus, submit
