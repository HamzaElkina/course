Exercice 7

🎯 Créer une fonction qui utilise une boucle, prompt, une condition et console.log

Écrivez une fonction interactive qui demande plusieurs nombres à l'utilisateur et effectue des calculs.

📌 Indications :
  • Créez une fonction nommée analyserNombres()
  • Utilisez prompt() pour demander combien de nombres l'utilisateur veut saisir
  • Utilisez une boucle for pour répéter la saisie autant de fois que demandé
  • Pour chaque nombre saisi :
    • Vérifiez avec une condition si le nombre est pair ou impair
    • Affichez dans la console : "Nombre X est pair/impair"
    • Calculez et stockez la somme de tous les nombres
  • Après la boucle, affichez dans la console : "Somme totale : X"
  • Affichez également la moyenne des nombres
  • Ajoutez un bouton HTML qui appelle cette fonction
💡 Objectifs détaillés :
  1. Demander avec prompt() : "Combien de nombres voulez-vous saisir ?"
  2. Stocker cette valeur dans une variable nombreDeValeurs
  3. Initialiser une variable somme = 0
  4. Créer une boucle for (let i = 1; i <= nombreDeValeurs; i++)
  5. Dans la boucle, demander à l'utilisateur de saisir un nombre avec prompt("Nombre " + i + " :")
  6. Convertir la saisie en nombre avec parseInt()
  7. Utiliser une condition if/else pour vérifier si le nombre est pair ou impair
  8. Afficher dans la console : "Nombre X est pair/impair"
  9. Ajouter le nombre à la somme
  10. Après la boucle, calculer la moyenne = somme / nombreDeValeurs
  11. Afficher dans la console : "Somme totale : " + somme et "Moyenne : " + moyenne
  12. Ajouter un bouton dans le HTML : <button onclick="analyserNombres()">Analyser des nombres</button>
📝 Structure à compléter dans votre HTML/JS :

<button onclick="analyserNombres()">Analyser des nombres</button>

<script>
    function analyserNombres() {
        // Étape 1 : Demander combien de nombres
        // Étape 2 : Initialiser somme = 0
        // Étape 3 : Boucle for pour les saisies
        // Étape 4 : Condition pair/impair
        // Étape 5 : Afficher dans la console
        // Étape 6 : Calculer somme et moyenne
    }
</script>
